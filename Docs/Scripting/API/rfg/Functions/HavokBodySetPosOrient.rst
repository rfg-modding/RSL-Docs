
HavokBodySetPosOrient
********************************************************
Sets the position and orientation of a havok rigid body.

**Arguments:**

Handle (`unsigned int`_)
    The handle of the havok rigid body.

Position (`Vector`_)
    The new position of the body.

Matrix (`Matrix`_)
    The new orientation matrix of the body.
