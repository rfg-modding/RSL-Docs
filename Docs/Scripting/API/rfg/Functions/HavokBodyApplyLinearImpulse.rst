
HavokBodyApplyLinearImpulse
********************************************************
Applies an impulse to a havok rigid body at it's center of mass.

**Arguments:**

Handle (`unsigned int`_)
    The handle of the havok rigid body.

LinearImpulse (`Vector`_)
    The impulse to be applied.
