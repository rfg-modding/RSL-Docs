
RemoveUserMessage
********************************************************
Remove a user message from the screen completely. Use `rfg.AddUserMessage`_ to create them. You should not use this messages handle after removing it. 

**Arguments:**

**Handle** (`int`_)
    The handle of the user message you want to remove.

**Returns:**

- None
