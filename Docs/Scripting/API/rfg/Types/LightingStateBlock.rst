LightingStateBlock
********************************************************
Lighting state values. Access this through ``rfg.Lighting``. So, for example, if you wanted to set the value of ``IndirectLightScale``, you'd do that like so:  ``rfg.Lighting.IndirectLightScale = 0.7``.
