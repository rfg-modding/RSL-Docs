SunShaftsStateBlock
********************************************************
Sunshaft/godrays state values. Access this through ``rfg.SunShafts``. So, for example, if you wanted to set the value of ``Scale``, you'd do that like so:  ``rfg.SunShafts.Scale = 1.2``.
