DofStateBlock
********************************************************
Depth of field (DOF) state values. Access this through ``rfg.Dof``. So, for example, if you wanted to set the value of ``FocusStartA``, you'd do that like so:  ``rfg.Dof.FocusStartA = 2.3``.
