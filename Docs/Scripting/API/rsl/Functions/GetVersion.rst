
GetVersion
********************************************************

**Arguments:**

- None

**Returns:**

**Version** (`String`_)
    Returns a string for the current RSL version. For example, the result for version 0.4.0 is ``0.4.0-Alpha``. This can be used to confirm that the user has a new enough version of RSL before running a script.
